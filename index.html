ADMIN PROTOTYPE

後台管理

Manager 可以新增、修改、刪除資料庫與文件；Owner 可以額外管理會員權限與查看操作紀錄。

Owner 最高權限：可管理資料、會員權限與操作紀錄。
Manager 管理權限：可新增、修改、刪除資料庫與文件。
User 查詢權限：可登入、查詢資料與使用小幫手。

新增測試文件

可新增文件、修改文件、移動所屬資料庫、設定標籤，並會寫入操作紀錄。
