ADMIN PROTOTYPE

後台管理

這裡是部門資料維護中心。總覽首頁僅供查看，真正的新增、修改與刪除請進入對應管理頁面操作。

Owner 部門維護權限：可管理資料、會員權限與操作紀錄。
Manager 資料維護權限：可新增、修改、刪除資料庫與文件。
User 使用權限：可登入、查詢資料與使用小幫手。
